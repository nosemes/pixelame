PIXELADOR FLASK

1) Activa l'entorn:
   conda activate cares

2) Instal·la Flask dins l'entorn:
   python -m pip install flask

3) Ves a la carpeta del projecte:
   cd C:\Users\nou\Desktop\flask_pixelador

4) Executa:
   python app.py

5) Obre al navegador:
   http://127.0.0.1:5000

Notes:
- Selecciona una foto de referència clara.
- Selecciona la carpeta amb les fotos originals.
- El resultat es descarrega com a ZIP.
- Tolerància recomanada: 0.50 a 0.60.
- Pixelació recomanada: 4 a 6.
